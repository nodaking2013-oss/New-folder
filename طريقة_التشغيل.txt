أهلاً بك في ZonaAI!

بما أننا أضفنا ميزات ذكاء اصطناعي حقيقية (مثل ChatGPT) للموقع، فنحن بحاجة إلى تشغيل برنامج صغير في خلفية الكمبيوتر (يسمى الباك إند أو Backend) ليتواصل مع الذكاء الاصطناعي بدلاً من الموقع مباشرة.

إليك أسهل خطوات لتشغيل كل شيء:

الخطوة 1: تنزيل لغة بايثون (Python)
----------------------------------
1. افتحي هذا الرابط: https://www.python.org/downloads
2. اضغطي على الزر الأصفر الكبير "Download Python".
3. بعد ما يتحمل، افتحيه للتثبيت.
📌 **مهم جداً جداً:** في أول شاشة تظهر لك في التثبيت، هتلاقي مربع صغير تحت مكتوب عليه "Add python.exe to PATH" .. لازم تحطي عليه علامة (علامة صح ✔) قبل ما تضغطي Install Now.

الخطوة 2: تثبيت الملفات المطلوبة
--------------------------------
بعد ما يخلص تثبيت البايثون:
1. افتحي مجلد المشروع بتاعك اللى اسمه (New folder) على الديسك توب.
2. جوه المجلد، اضغطي كليك يمين فى أي مكان فاضي واختاري "Open in Terminal".
3. هيفتح شاشة سوداء، انسخي الكود ده وحطيه فيها واضغطي انتر:
pip install -r requirements.txt
(الكود ده بينزل الحاجات اللى بتخلي البايثون يكلم الذكاء الاصطناعي).

الخطوة 3: الحصول على الرقم السري (API Key)
------------------------------------------
عشان الموقع بتاعك يشتغل مجاناً، لازم رقم سري من جوجل:
1. ادخلي على الرابط ده: https://aistudio.google.com/
2. سجلي دخول بحساب جوجل بتاعك (الجيميل).
3. اضغطي على زرار "Get API key" أو "Create API Key" باللون الأزرق.
4. انسخي الكود الطويل اللى هيطلعلك.
5. افتحي ملف `backend.py` اللى فى المشروع، وهتلاقي السطر ده:
API_KEY = "YOUR_GEMINI_API_KEY_HERE"
امسحي كلمة YOUR_GEMINI_API_KEY_HERE وحطي الرقم السري اللى نسختيه من جوجل (بين علامتين التنصيص).

الخطوة 4 والأخيرة: تشغيل الموقع بشكل كامل!
-------------------------------------------
1. في نفس الشاشة السوداء (Terminal) اللى فتحتيها فى الخطوة التانية، اكتبي الكود ده واضغطي انتر:
python backend.py
2. هيبدأ البرنامج يشتغل ويكتبلك "ZonaAI Python Backend Server Is Running!".

خلاص كده! سيبي الشاشة السوداء دي شغالة متقفليهاش.. وافتحي ملف `index.html` فى المتصفح واضغطي على أي ميزة زي ChatGPT وهتلاقيها شغالة وبترد عليكي بجد!
